## 📦 Tech Stack

- **Frontend**: React, CSS Modules, AOS (Animate on Scroll)
- **Forms**: Formspree (email integration)
- **Icons**: React Icons
- **Deployment**: Netlify
- **SEO & Accessibility**: Semantic HTML, ARIA labels, responsive design
